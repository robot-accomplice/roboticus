## LinkedIn (3,000 characters max)

```
Roboticus v1.0.0: The Go Rewrite

After months of development, I'm releasing Roboticus v1.0.0 — a complete ground-up rewrite of our autonomous AI agent runtime, from Rust to Go.

This isn't an incremental update. It's 140,000 lines of new Go code achieving exact feature parity with the Rust v0.11.4 reference, verified by a 101-gap audit where every divergence was tracked to closure.

What ships in v1.0.0:

Runtime
- ReAct agent loop with 25-guard output safety pipeline
- Semantic classifier with 5 exemplar categories for behavioral defense
- 4-layer personality system (OS, FIRMWARE, OPERATOR, DIRECTIVES)
- Consolidation engine with 5-tier memory (Working, Episodic, Semantic, Procedural, Relationship)

Intelligence
- 10+ LLM providers (OpenAI, Anthropic, Gemini, Ollama, OpenRouter, and more)
- 6-axis metascore routing with ML preference learning
- 3-tier semantic cache (exact hash, tool-aware TTL, cosine similarity)
- Hybrid FTS5 + vector search for memory retrieval

Channels
- 9 adapters: Telegram, Discord (full WebSocket gateway), Signal, WhatsApp, Email (OAuth2), Voice, Matrix (E2E encrypted), Agent-to-Agent (X25519+AES), Web
- MCP client/server with stdio + SSE transports

Infrastructure
- 38 CLI commands for full operator control
- Embedded dashboard with 6-axis routing profile editor
- EVM wallet with EIP-3009, x402 payments, and treasury policy
- Plugin system with managed registry
- Pure Go, no CGO (modernc.org/sqlite), cross-compiles to 5 platforms

Quality
- 329 test files, 3,232 test functions, 24/24 packages passing
- Live MCP integration tested with Playwright
- golangci-lint clean, govulncheck clean
- Behavioral fitness test suite

Why rewrite? Go's deployment simplicity (single static binary), ecosystem maturity for network services, and the ability to cross-compile without toolchain complexity. The Rust version was excellent engineering — the Go version matches it feature-for-feature and ships as a drop-in replacement.

The parity methodology was rigorous: every gap was fixed to match Rust behavior exactly. No improvisation, no divergence. Port first, improve later.

Install: curl -fsSL https://roboticus.ai/install.sh | sh

More at https://roboticus.ai
```
