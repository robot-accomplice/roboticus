# v1.0.3 Release Notes

**Date**: 2026-04-12
**PRs**: #32, #33, #34
**Tag**: `v1.0.3`

---

## Summary

WebSocket-first dashboard architecture replacing all HTTP polling, struct-driven
settings schema derived from the Config struct, plugin/app catalog with registry
fallback, and a sweep of workspace/theme/layout bug fixes. Zero `setInterval`
calls survive in the dashboard.

## Changes

### WebSocket-First Dashboard (PR #34)

| Component | Description |
|-----------|-------------|
| `ws_protocol.go` | WebSocket upgrade, ticket validation, topic subscription protocol |
| `ws_topics.go` | Topic registry: sessions, traces, health, logs, metrics, config |
| Pipeline bridge | Pipeline publishes lifecycle events to EventBus; WS subscribers receive them |
| Ticket auth | WS connections require a validated ticket (anti-CSRF, anti-replay) |
| Polling removal | All `setInterval`-based polling replaced with topic subscriptions |

Also: concurrency rationale documented for the WebSocket-first security model.

### Struct-Driven Settings Schema (PR #33)

| Component | Description |
|-----------|-------------|
| `config_schema.go` | `/api/config/schema` endpoint — 303 fields reflected from Config struct |
| `config_defaults.go` | Default values extracted from `DefaultConfig()` via reflect |
| `config_loader.go` | TOML + env overlay loader with validation |
| `config_validation.go` | Constraint enforcement (ranges, enums, required fields) |
| Settings UI | Dashboard settings page now derives form from schema, not hardcoded TOML |

### Plugin + App Catalog (PR #33)

- Remote plugin catalog from `roboticus.ai/registry/plugins.json`
- Registry fallback: local installed plugins merged with remote catalog
- Installed-only theme dropdown (no phantom entries)
- Fake/planned plugins removed from catalog — only real ones listed

### Workspace / Layout Fixes (PRs #32, #33)

| Fix | Description |
|-----|-------------|
| Footer pinning | Workspace footer pinned via `height:100%` — was `calc()` misfire |
| Workstation layout | Equidistant positioning with dynamic edge clamping |
| Canvas sizing | `resize()` controls dimensions — removed conflicting CSS `width`/`height` |
| `isAgent` crash | Null guard on workspace agent detection |

### Theme / Rendering Fixes (PRs #32, #33)

| Fix | Description |
|-----|-------------|
| `_catalogThemeVars` crash | Undefined variable in Skills & Plugins page — now guarded |
| `parseThemeColors` | Cached per frame, invalidated on theme change only |
| Theme serialization | Catalog now serializes variables/textures/fonts for preview rendering |
| Duplicate theme | Duplicate psychedelic theme entry removed |
| Enriched themes | All catalog entries carry full theme metadata |
